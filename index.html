Treasure Hunt In Forest

Adventure

You are in a forest to find the treasure. Choose the options to navigate and find the treasure. If you want to head back, choose that option to navigate. You may encounter various creatures as you navigate further. Choose the directions carefully to avoid trouble.

Addendum

  • This a interactive story explained using picture
  • To design the adventure we need to create html file
  • The html file contains the basic layout of the page and also contains linking javascript and css files
  • The css file contains the font styles and background color
  • There is #text-container and #story-container block which are used to sent respective formats for them
  • The is cointainer object that is being defined in javascript file which explains various plots
  • The background color and image size are set of all the containers using that object
  • The java script file consist of various functions
  • Each object will have its text image and the choices for the user to enter
  • By clicking the choices at each phase the button will be redirecting to the repective page
  • There is Update() function which is being defined to update the page based on user choice
  • The function of the each button will be discribed in the javascript file
  • The start() function is called at the end in javascript file to start the adventure
  • Respective images that are being linked are stored in same folder to display
  • End() function is used to end the adventure
